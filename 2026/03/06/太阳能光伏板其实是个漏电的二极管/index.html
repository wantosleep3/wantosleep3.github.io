太阳能光伏板其实是个‘漏电的二极管’

2026-03-06

很多人把太阳能板当成普通恒压电源。这完全偏离了物理本质。太阳能板的核心是超大面积PN结。它是一个受光照控制的电流源并联了一个漏电二极管。

太阳能板等效模型

I=Iph−Is(eVnVt−1)

I 代表面板最终对外的实际输出电流
Iph 代表太阳光照激发出来的光生电流
减号后面那一整代表内部二极管正向导通漏掉的电流
Is 是PN结二极管的反向饱和电流
V 是面板当前的输出电压
n 是二极管的理想因子
Vt 是物理学里的热电压常数

I=Iph−Id

Iph 是光生电流
Id 是内部二极管分流
Is(e…−1) 就是这个 Id

直接看等效输出电流公式。输出电流 I 等于光生电流 Iph 减去二极管电流 Id 。光照产生电子形成 Iph 。内部PN结正向偏置会吃掉一部分电流形成 Id 。在低输出电压区域工作时内部二极管几乎不导通。漏掉的电流极小。输出电流近似等于恒定的光生电流。太阳能板在大部分区间表现得像个电流源。

内部二极管导通机制

输出电压继续升高会发生什么变化？内部二极管开始强烈导通。二极管漏电流呈指数级暴涨。

输出电压接近开路电压 Voc 时漏回内部的电流等于光生电流。对外输出电流直接归零。这就是为什么I-V曲线断崖式下跌

光照衰减与对数效应

傍晚光照锐减百分之九十。总电流降低吗？肯定会降低。光照减弱导致光生载流子成比例减少。输出电流直接掉到原来的十分之一。总电压会不会大幅降低？并不会大幅降低。根据二极管公式可以反推开路电压。

Voc=nVtln⁡(Iph/Is+1)

公式里光生电流 Iph 处于自然对数 ln 内部。对数函数具有极强抹平效应。 Iph 暴跌百分之九十后对数值只减去一个很小常数。实际面板仅仅出现两三伏电压波动。光照减少一大半后太阳能板开路电压依然能保持

最大功率点工作逻辑

最大功率点电压 Vmp 是不是固定了电流？并不是这样。很多人套用公式 P=UI 觉得锁死电压就能锁死电流。 Vmp 只是当前光照条件下功率最大值对应的电压坐标。光照变弱时整条曲线向下平移。
MPPT模块在新的曲线里会重新寻找最高点。它通过微调电压迎合此时极其有限的电流。 Vmp 是最终寻优得到的结果。它没法反过来决定电流大小。这次我们成功回顾了对数公式和内部二极管模型
